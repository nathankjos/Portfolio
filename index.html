Browser Organizer

My goal for this project was to create an organizational tool to replace my homepage with. I wanted to have an organized browser sorted by categories that I could create, have each category have a to-do list reminding me of what I would like to get done specific to each category, have a list of links saved to easily access, and a notes section to write down any thoughts that I might want to reference later.

Expense Tracker

I wanted to create a more specialized expense tracker for those who live paycheck to paycheck. One that had features to help highlight important expenses that are not optional, and show the remaining funds available that they have to spend, or how much more they need to save in order to be able to pay for everything. I also wanted to emphasize the importance of having an emergency fund especially for those in these situations, so that if something does happen, they would be in a better position to deal with issues that may arise

Technologies Used: -Ruby on Rails -JQuery -HTML5 -Javascript(ES6) -CSS3 -Mongoose
